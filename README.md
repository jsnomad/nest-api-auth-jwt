## Setup

Clone the repo then install dependencies :

```bash
npm install
```

### Start

Run the following command to start the HTTP server (dev mode).

```bash
npm start
```

Create .env file :

```bash
cp .env.example .env
```

### Swagger Documentation

Start the server, then

```bash
http://localhost:3001/docs/
```
